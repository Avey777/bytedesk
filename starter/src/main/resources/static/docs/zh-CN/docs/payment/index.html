版本与价格​

微语系统提供多种版本选择，满足不同规模企业需求：

版本价格主要特点
社区版免费开源，可商用，单租户，社区版源码，保留源码版权声明，用户数不限，客服/机器人/知识库各限50个。
企业版3万元单租户，功能完整，私有化部署包，不含源码，用户数/客服/机器人/知识库数量不限，支持品牌自定义
平台版6万元多租户，SaaS，功能完整，私有化部署包，不含源码，用户数/客服/机器人/知识库数量不限，支持品牌自定义
产品代理企业版8折适用于购买企业版≥2份授权，私有化部署包，不含源码
源码版2万元起包含完整源码，支持按模块单独购买。含平台版所有功能，无需再购买企业版或平台版，所购买模块无需授权licenseKey。区别于企业版：可以自行修改源码，支持定制化开发。

注：价格为一次性收费

租户说明：一个租户代表一个企业账号，一个企业账号下可创建多个部门、成员和客服账号。不同租户数据相互隔离，互不影响。
